# E-Book Generator

## Overview
E-Book Generator is a full-stack MERN application that allows users to create and manage e-books through a user-friendly interface.

## Features
- User Authentication
- Create, Edit, and Delete E-Books
- Responsive Design
- REST API Integration
- MongoDB Database

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- HTML
- CSS
- JavaScript
